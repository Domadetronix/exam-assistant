# Exam Assistant — PWA для подготовки к экзамену

Офлайн-приложение для подготовки к экзамену: теоретические билеты, режим тестирования и практические справочники.

## Стек

- React 18 + TypeScript + Vite
- React Router
- Zustand (состояние)
- IndexedDB через `idb`
- vite-plugin-pwa (Service Worker + manifest)

## Возможности

### Теория
- Список билетов с категориями и поиском
- Поиск по вопросам и ответам с приоритетом (вопрос → тег → ответ)
- Фильтрация по категориям
- Карточка вопроса (модальное окно) с тегами и связанными вопросами
- Гибридный режим связанных вопросов: ручные `related`, иначе авто-подбор по категории и тегам

### Тест
- Случайный вопрос с blur-ответом
- Confirmation-диалог перед раскрытием
- Кнопка «Следующий вопрос»

### Практика
- Справочники по направлениям: Blender, Adobe Animate, Дизайн публикаций, Дизайн мобильных приложений, Веб-разработка
- Безопасный markdown-рендер (только React-узлы, никаких сырых HTML-вставок)

### PWA
- Манифест и SVG-иконки
- Service Worker с кэшированием статики и изображений (autoUpdate)
- Установка на телефон как обычное приложение
- Полностью офлайн после первого запуска

### Настройки
- Импорт билетов из JSON или текстового файла (формат `Q:` / `A:`)
- Экспорт текущей базы в JSON
- Очистка данных приложения (IndexedDB, LocalStorage, Cache API, Service Worker)

## Установка и запуск

```bash
npm install
npm run dev
```

Сборка:

```bash
npm run build
npm run preview
```

## Импорт данных

Поддерживаются два формата:

**JSON** — массив объектов `Ticket`:

```json
[
  {
    "id": 1,
    "category": "Теория цвета",
    "tags": ["цвет"],
    "question": "Что такое цветовой круг?",
    "answer": "Цветовой круг — это..."
  }
]
```

**Текст / Markdown:**

```
# Категория: Теория цвета
## Tags: цвет, дизайн

Q: Что такое цветовой круг?
A: Цветовой круг — это схематическое представление цветов спектра...

Q: Какие бывают цветовые схемы?
A: Монохромная, аналоговая, комплементарная...
```

Категория и теги действуют до следующего объявления.
